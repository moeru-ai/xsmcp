## [srvx](https://srvx.h3.dev)

When you use `srvx`, your code can also run on Deno and Bun.

```package-install
npm i srvx @xsmcp/server-http @xsmcp/server-shared
```

```ts twoslash
import { fetch } from '@xsmcp/server-http'
import { createServer } from '@xsmcp/server-shared'
import { serve } from 'srvx' // [!code highlight]

const server = createServer({
  name: 'example-server',
  version: '1.0.0'
})

// https://srvx.h3.dev/guide/options
const server = serve({
  fetch: fetch(server),
  port: 3000, // [!code highlight]
});
```

### srvx/node

Use when you only want Node.js compatibility.

```diff
- import { serve } from 'srvx'
+ import { serve } from 'srvx/node'
```
